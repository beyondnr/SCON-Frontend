## 배경 (Context)
GitHub CLI(`gh`)를 활용하여 로컬 Markdown 파일로부터 이슈를 자동 생성하고, 프로젝트 로드맵에 일정을 등록하는 자동화 프로세스를 테스트합니다.

## 목표 (Goals)
- `gh` 명령어를 통해 이슈가 정상적으로 생성되는지 확인
- 생성된 이슈가 GitHub Project에 올바르게 추가되는지 검증
- 시작일(Start Date)과 종료일(Target Date)이 정상적으로 설정되는지 확인

## 작업 상세 (Tasks)
- [ ] 테스트 이슈 파일(`ISSUE-99-test-issue.md`) 생성
- [ ] PowerShell 스크립트를 통해 이슈 생성 및 프로젝트 등록 실행
- [ ] GitHub 웹사이트에서 생성된 이슈 및 로드맵 일정 확인
- [ ] 테스트 완료 후 이슈 닫기(Close)

## 참고 사항 (Notes)
이 이슈는 테스트 목적으로 생성되었으며, 검증 후 삭제되거나 닫힐 예정입니다.
